# Blue Green Deployment

Create the systemd service:

```bash
chmod +x create-systemd-service.sh
sudo ./create-systemd-service.sh
```

Create the nginx configuration:

```bash
chmod +x create-nginx-sites.sh
sudo ./create-nginx-sites.sh
```
